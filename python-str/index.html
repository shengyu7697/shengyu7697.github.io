Python 字串基本用法

Python 的字串語法為文字前後加上兩個單引號 '' 或兩個雙引號 ""，例如：'hello world' 或者 "hello world"，依個人喜好或情境來選擇使用，例如今天如果遇到字串裡有包含 ' 像是 "ShengYu's notebook"，就使用兩個雙引號來表示字串。

Python 字串的變數類型為 str，可以使用 type() 來檢查變數類型得知。
