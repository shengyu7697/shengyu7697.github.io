Ubuntu 下安裝 wordpress

本篇 ShengYu 紀錄在 Ubuntu 下安裝 wordpress，我自己是用 Ubuntu 22.04 安裝 wordpress 6.5.5。

下載wordpress-6.5.5版

或者下載最新版

新建資料庫

編輯 wordpress.conf
sudo vim /etc/apache2/sites-available/wordpress.conf

重啟 apache

開啟網頁，填上剛剛稍早設定的資料，按下安裝，搞定！
資料庫名稱 wordpress
使用者名稱 wordpressuser
密碼 your-password
