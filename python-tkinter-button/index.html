Python tkinter Button 按鈕用法與範例

以下的 Python tkinter Button 用法與範例將分為這幾部分，

  • tkinter 建立按鈕 Button
  • tkinter 觸發按鈕事件來修改按鈕文字
  • tkinter Button 搭配 lambda 運算式寫法
  • tkinter 改變按鈕顏色
  • tkinter 改變按鈕大小
  • tkinter 按下按鈕離開程式
