32/64bit дҪңжҘӯзі»зөұ printf еҲ—еҚ° int64_t / uint64_t зҡ„ж–№жі•

жң¬зҜҮе°Үд»Ӣзҙ№еңЁ C/C++ зЁӢејҸиЈЎеҰӮдҪ• printf еҲ—еҚ°еҮә int64_t жҲ– uint64_t йҖҷеҖӢи®Ҡж•ёйЎһеһӢпјҢ

int64_t еңЁ Linux 64-bit дҪңжҘӯзі»зөұ printf еҲ—еҚ°иҰҒз”Ё %ld
int64_t еңЁ Linux 32-bit дҪңжҘӯзі»зөұ printf еҲ—еҚ°иҰҒз”Ё %lld
uint64_t еңЁ Linux 64-bit дҪңжҘӯзі»зөұ printf еҲ—еҚ°иҰҒз”Ё %lu
uint64_t еңЁ Linux 32-bit дҪңжҘӯзі»зөұ printf еҲ—еҚ°иҰҒз”Ё %llu
int64_t еңЁ Windows 64-bit/32-bit дҪңжҘӯзі»зөұ printf еҲ—еҚ°иҰҒз”Ё %lld
uint64_t еңЁ Windows 64-bit/32-bit дҪңжҘӯзі»зөұ printf еҲ—еҚ°иҰҒз”Ё %llu

int64_t жҲ– uint64_t жңғж №ж“ҡ 32/64-bit дҪңжҘӯзі»зөұдёҚеҗҢиҖҢе®ҡзҫ©дёҚеҗҢйЎһеһӢпјҢ

Pythonж•ҷеӯё

[Python] дҪҝз”Ё pyinstaller е°ҮзЁӢејҸжү“еҢ…жҲҗ exe зҷјдҪҲ

жң¬зҜҮд»Ӣзҙ№еҰӮдҪ•дҪҝз”Ё pyinstaller е°Ү python зЁӢејҸжү“еҢ…жҲҗ exe еҹ·иЎҢжӘ”зҷјдҪҲпјҢеӯёжңғйҖҷжӢӣеҫҢе°ҮеҸҜд»ҘжҠҠ exe еҹ·иЎҢжӘ”жӢҝеҲ°д»»дҪ• windows йӣ»и…ҰиЈЎеҹ·иЎҢпјҢеҚідҪҝйӮЈеҸ° windows йӣ»и…ҰжІ’жңүиЈқ python з’°еўғд№ҹеҸҜд»Ҙеҹ·иЎҢе”·пјҒ

е®үиЈқ pyinstaller

pyinstaller жҳҜдёҖеҖӢж–№дҫҝзҡ„ python жЁЎзө„пјҢе°Ү python зҡ„ .py жӘ”жү“еҢ…жҲҗзӮә exe еҹ·иЎҢжӘ”пјҢйҒҺзЁӢжңғе°ҮжүҖжңүзҡ„зӣёдҫқжЁЎзө„дёҖдҪөжү“еҢ…гҖӮ
еҰӮжһңжІ’жңү pyinstaller, е…Ҳз”Ё pip е®үиЈқ

йӣўз·ҡе®үиЈқ pyinstaller

еҲ°PyPIдёӢијүпјҢзӣ®еүҚжІ’жңүжҸҗдҫӣ.whlжӘ”дёӢијүпјҢ
д»Ҙзӣ®еүҚдёӢијүеҲ°зҡ„PyInstaller-3.6.tar.gzзӮәдҫӢеӯҗзҡ„и©ұпјҢијёе…ҘдёӢеҲ—pipжҢҮд»Өе°ұжңғеҺ»е®үиЈқpyinstaller

еҫҲеҸҜжғңең°дјјд№Һе®үиЈқйҒҺзЁӢдёӯжңғжңүе…¶д»–зӣёдҫқзҡ„еҘ—д»¶пјҢpipи©Ұең–йҖЈз¶Іе®үиЈқзјәе°‘зҡ„зӣёдҫқеҘ—д»¶пјҢзөҗжһңжңҖеҫҢеӣ зӮәйӣўз·ҡй—ңдҝӮжңғйҖЈз·ҡйҖҫжҷӮ

еҲ°conda-forgeдёӢијү pyinstaller-3.6-py37h7602738_0.tar.bz2
з”Ё conda жҢҮд»Өе®үиЈқпјҡ

д»ҘзӮәз”Ё conda-forge зҡ„йӣўз·ҡеҢ…е°ұиғҪиЈқе®Ңе…ЁйғЁзӣёдҫқеҘ—д»¶пјҢ
жңҖеҫҢзҷјзҸҫйӮ„жҳҜе°‘дёҖдәӣеҘ—д»¶пјҢеҸӘеҘҪиӘҚе‘Ҫзҡ„жҠҠзјәе°‘зҡ„иЈқдёҖиЈқдәҶпјҢ
еҲҶеҲҘжҳҜ

  • pywin32-ctypes
  • pefile
  • altgraph

иЈқе®ҢеҫҢе°ұзҷјзҸҫ pyinstaller жҢҮд»ӨеҸҜд»ҘдҪҝз”ЁдәҶпјҒ

дҪҝз”ЁзҜ„дҫӢ

дҪҝз”Ёpyinstallerжү“еҢ…жҲҗexeжңҖз°Ўе–®зҡ„зҜ„дҫӢеҰӮдёӢжүҖзӨәпјҢй җиЁӯжҳҜжңғжү“еҢ…жҲҗдёҖеҖӢиіҮж–ҷеӨҫпјҢ
pyinstaller жңғеҺ»еҲҶжһҗзЁӢејҸдёӯжүҖз”ЁеҲ°зҡ„ moduleпјҢжңҖеҫҢжҠҠйҖҷдәӣжқұиҘҝж”ҫеңЁдёҖеҖӢиіҮж–ҷеӨҫжҲ–е–®дёҖеҹ·иЎҢжӘ”(зңӢдҪ дҪҝз”Ёзҡ„еҸғж•ё)
д»ҘйҖҷеҖӢзҜ„дҫӢдҫҶиӘӘпјҢжңғжү“еҢ…жҲҗдёҖеҖӢyourprogramиіҮж–ҷеӨҫпјҢyourprogramиіҮж–ҷеӨҫиЈЎйқўжңғжңүдёҖеҖӢyourprogram.exe

дҪҝз”Ёpyinstallerжү“еҢ…жҲҗдёҖеҖӢеҹ·иЎҢжӘ”exeпјҢдёҰдё”дёҚйЎҜзӨә cmd иҰ–зӘ—пјҢ

Gitж•ҷеӯё

Git жҜ”е°ҚйҖҷж¬Ў commit и·ҹеүҚдёҖж¬Ў commit зҡ„е·®з•°

йҖҷз®—жҳҜеҫҲйҖІйҡҺгҖҒеҫҲж–№дҫҝзҡ„дёҖжӢӣ, жңүжҷӮеҖҷдёҚе°ҸеҝғжҠҠеһғеңҫжҲ–йҷӨйҢҜз”Ёзҡ„ code зөҰдёҖиө· commit йҖІеҺ»дәҶ,
йңҖиҰҒжҠҠйҖҷдәӣжқұиҘҝз§»йҷӨжҺү,
дҪҶжҳҜе“Әж®өжүҚжҳҜжғіиҰҒеҺ»йҷӨзҡ„йӮЈж®өе‘ўпјҹ
йҖҷжҷӮеҖҷеҸҜд»Ҙз”ЁдёӢеҲ—жҢҮд»ӨжҜ”е°ҚеҮәйҖҷж¬Ў commit и·ҹеүҚдёҖж¬Ў commit зҡ„е·®з•°,

д№ӢеҫҢе°ұеңЁеӨ–йғЁжҜ”е°Қе·Ҙе…·(meld)иЈЎйҒёж“ҮиҰҒз§»йҷӨзҡ„ code
д№ӢеҫҢеҶҚ git add з„¶еҫҢ git commit --amend е°ұжҗһе®ҡдәҶпјҒи¶…ж–№дҫҝзҡ„жҠҖе·§пјҒ
