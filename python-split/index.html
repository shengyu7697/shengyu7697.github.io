Python split 字串切割使用範例

Python split 的用處在於把字串透過指定的分隔符號對字串進行分割，分割後的字串會儲存在一個串列 list，
split() 第一個參數為指定的分隔符號，可以是空白,換行(\n)，tab(\t), 逗號(,) 或其他字元等等，
split() 不給入參數的話，預設會以空白作為分隔符號做切割，
