開源專案-金庸群俠傳

今天介紹Github上開源專案的金庸群俠傳遊戲，

scarsty/kys-cpp

https://github.com/scarsty/kys-cpp
星星數：168stars
技術：Cpp,SDL,sqlite3,boost asio
使用 C++ 與 SDL2 的「金庸群俠傳」復刻版。
