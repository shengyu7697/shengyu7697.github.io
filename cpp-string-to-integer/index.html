C/C++ 字串轉數字的4種方法

本篇介紹 C/C++ 字串轉數字 string to integer 的方法，總共將會介紹4種方法，

以下 C/C++ 字串轉數字的4種方法分別是，

  • C 的字串轉數字 atoi()
  • C++ 字串轉數字 std::atoi()
  • C++11 的字串轉數字 std::stoi()
  • 自製 atoi()
