Python 2to3 - 將 Python 2 程式碼轉為 Python 3 程式碼的工具

Python 2to3 它可以在命令行中使用 將 Python 2.x 程式碼轉換成 Python 3.x 版本的程式碼，

這個指令會印出和原始碼的區別。透過傳入 -w 參數，

2to3 也可以把需要的修改寫回到原檔案中（除非傳入了 -n 參數，否則會為原始檔案建立一個副本）。
