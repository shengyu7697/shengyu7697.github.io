在 Ubuntu 下安裝 OpenShot

今天介紹一套免費 、好用、跨平台的剪輯軟體 OpenShot！
介紹一下怎麼在 ubuntu上 安裝

安裝環境為 Ubuntu 16.04
選擇使用 PPA 安裝
(試過下載官方提供的 OpenShot-v2.4.1-x86_64.AppImage 檔可以執行, 但是語言選擇卻沒有繁體中文)

顯示一下安裝的版本
