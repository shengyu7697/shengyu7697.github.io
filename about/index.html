關於我

大家好！我是 ShengYu。

我喜歡記錄我學習過的事物，過去在 Windows / Linux / macOS 平台上有豐富的寫程式經驗，一路上的跌跌撞撞的學習經驗希望可以讓大家少走點彎路，主要使用的程式語言為 C/C++、Python、Java，常用函式庫有 OpenCV、Qt，關注於影像處理、底層技術、嵌入式系統和開源專案，同時我最喜歡寫程式來輔助解決工作與生活上的問題。

我的興趣很多元，主要的有桌遊、閱讀、研究投資理財、旅遊、美食、唱歌，
上班族朝九晚五的生活過膩了，突然某天挑戰獨自一人熱血騎機車環島當起背包客，這趟旅程讓我更認識了台灣，這些經歷與認識結交的朋友讓我獲得了許多人生啟發，也從此在心中萌起了環遊世界的夢想，
同時我是一個團長，過去曾經辦過桌遊、唱歌、騎單車等活動聚會，
我很熱衷於日本旅行，每年都會安排1~2次日本自助旅行，喜歡找出房費CP值最高日期區間出國玩，因此還寫了一個程式來輔助訂房。

我擅長解決問題也喜歡幫助別人解決在學習寫程式上遇到的困難，
目前接受一對一咨詢，輔導程式教學，主要內容有 Python、C/C++。
