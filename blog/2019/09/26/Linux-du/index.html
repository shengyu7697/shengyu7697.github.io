如何使用 Linux 指令，du 查詢硬碟剩餘空間/資料夾容量

本篇記錄如何使用 Linux 下的 du 指令來查詢硬碟剩餘空間。

查詢當下目錄的每個資料夾總共佔用硬碟多少(-d1 代表深度為1)
du -d1 -h

相關方法
使用 ubuntu 桌面版也可使用disk工具來查詢硬碟剩餘空間。
