glm 安裝

OpenGL Mathematics (GLM) 是由C++寫成的標頭檔數學函式庫，是基於 OpenGL Shading Language（GLSL）規範的圖形軟體。

安裝

從官網這裡下載，
直接 include 即可，不須編譯。

查看 glm version 版本號

GLM_VERSION 定義在 glm/detail/setup.hpp
