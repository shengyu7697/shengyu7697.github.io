在 Ubuntu 安裝 StarUML

本篇介紹一款在 Ubuntu 下 UML 的軟體「StarUML」，StarUML 是一個開源的 UML 工具，用 nodejs 寫成，目前永久免費試用。

推薦安裝 2.8 (輸出JPEG圖檔沒有浮水印)，可以透過下面官網去下載，
網路上也一些 3.x 的破解方式，有興趣的話可以去找看看。
官網下載：http://staruml.io/download
支援平台：macOS 10.9 以上 / Windows 7 以上 / Linux

(開源？不過去我去官網找不到源碼連結)

Ubuntu 16.04 安裝方式

libgcrypt11 可以透過 下面網址下載
https://launchpad.net/ubuntu/+archive/primary/+files/libgcrypt11_1.5.3-2ubuntu4.2_amd64.deb

Ubuntu 18.04 安裝方式

Ubuntu 18.04 安裝方式可以參考 https://stackoverflow.com/a/50538353
