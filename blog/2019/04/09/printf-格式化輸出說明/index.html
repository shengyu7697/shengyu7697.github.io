printf 格式化輸出說明

以下列出常用的格式化輸出參數與說明︰

字元/字串 說明
%c 字元(char)
%s 字元陣列(char *)
整數 說明
%d、%i 十進制整數(int)
%ld、%Ld 十進制長整數(long)
%hd 十進制短整數(short)
%x 十六進制
%ld (int64_t => long int linux 64bit)
%lld (int64_t => long int linux 32bit)
%lu (uint64_t => unsigned long int linux 64bit)
%llu (uint64_t => unsigned long int linux 32bit)
%I64d (long long)
%I64u (unsigned long long)
%I64x 64bit的16進制
浮點數 說明
%f 單精度浮點數，預設輸出精度6位(float)
其他 說明
%p 指標
