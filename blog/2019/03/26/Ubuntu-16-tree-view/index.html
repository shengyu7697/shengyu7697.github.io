Ubuntu 16.04 檔案總管使用樹狀顯示方式

使用樹狀顯示的好處是移動檔案很快速很方便，
直接一直點開資料夾展開樹狀然後選到要移動的檔案，
原本的顯示方式需要一直雙擊進入資料夾，相比之下，樹狀顯示顯得比較快速，
在 Ubuntu 16.04 有提供這功能，但預設沒有開啟，

以下介紹怎麼開啟樹狀顯示功能︰
nautilus - Edit > Preferences > Display > List View
勾選 Navigate folders in a tree 即可生效
