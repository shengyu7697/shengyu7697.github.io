Linux 比 grep 還快的 ag 搜尋指令

本篇介紹 Linux 下超強的搜尋字串工具「The Silver Srarcher」，The Silver Srarcher 簡稱 ag, ag 號稱比 ack 快(ack 又號稱比 grep 快)，這麼厲害的 ag 搜尋工具還不趕緊手刃在 Linux 裝起來試試。
Ubuntu 下安裝方法為 sudo apt-get install silversearcher-ag

假設要搜尋 main 這個關鍵字
ag "main\"

假設要搜尋 main ( 這個關鍵字，( 搜尋特殊字元要加 \
ag "main\("

指定 src 目錄下搜尋
ag "main\(" src

排除 3rdparty 目錄之外
ag "main\(" --ignore-dir 3rdparty

排除 3rdparty, doc 目錄之外
ag "main\(" --ignore-dir 3rdparty --ignore-dir doc

另外其他很有用的選項是
ag -i 使用不敏感的匹配方式
ag -w 全詞匹配
ag -G ".+\.java" 搜索 java 類型的檔案
ag -l 顯示有匹配的檔案路徑
ag -L 顯示沒有任何匹配的檔案路徑
ag -v 反向匹配，將匹配到的資料排除
