如何使用 Linux tmux 指令 - 終端機管理工具

本篇將介紹如何使用 Linux 下的 tmux 終端機管理工具，

分割視窗

左右垂直分割：Ctrl+b %
上下水平分割：Ctrl+b "
分割完後要切換分割視窗要按 Ctrl+b <方向鍵上/下/左/右>
