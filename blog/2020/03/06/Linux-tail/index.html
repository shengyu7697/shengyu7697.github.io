如何使用 Linux 指令，tail 持續監看檔案輸出

本篇將介紹如何使用 Linux 下的 tail 指令看檔案內容的最後幾行或持續監看檔案輸出，並簡單介紹 tail 用法與範例。

範例

tail 會讀取檔案的所有內容並將檔案內容最後部份作為標準輸出，預設輸出10行。
-f: 循環讀取，持續監看最新追加的內容，常用於查閱正在改變的日誌文件。

監看持續成長的紀錄檔，這樣的指令組合相當於dmesg。
