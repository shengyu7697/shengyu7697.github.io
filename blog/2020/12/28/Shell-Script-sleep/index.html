Shell Script sleep 延遲執行

Shell Script 中偶爾需要延遲 sleep 幾秒鐘再繼續執行程式或腳本這個功能，例如我想要間隔1秒取得CPU使用量等等諸如此類的場合應用，本篇就介紹如何在 Shell Script 中使用 sleep 指令延遲或暫停，然後再繼續執行。

如果答案只能這樣的話未免也令人太灰心了吧！
另一種人性化的答案就是將前面的招式融會貫通組合起來，

總結

整理一下 sleep 後面接受的時間單位，
語法：sleep NUMBER[SUFFIX]...

  • s：秒 seconds (預設)
  • m：分鐘 minutes.
  • h：小時 hours.
  • d：天 days.

後記

macOS 下的 sleep 指令只能接受秒的參數
