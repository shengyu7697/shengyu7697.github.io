C/C++ 3種字串轉數字的方法與範例

本篇介紹 C/C++ 字串轉數字 string to integer 的方法，總共將會介紹3種方法，

C/C++ 字串轉數字 string to int 的分別是

  • 第一種是標準 C 的 atoi() 函式
  • 第二種是 C++ 的 std::atoi() 函式
  • 第三種是 C++11 的 std::stoi() 函式
