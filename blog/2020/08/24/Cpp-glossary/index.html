C/C++ 一堆名詞解釋

紀錄一下 C/C++ 一堆名詞解釋。

覆載 override

就是子類覆蓋掉父類的實作叫覆載 override，就像例子中的bbb

多載 overload

就是相同函式名，但參數不一樣，叫多載 overload

其他雜類

繼承：inheritance
多型：polymorphism
建構子：constructor
複製建構子：copy constructor
解構子：destructor
函子/函式物件：functor
抽象類別：abstract class
父類：base class
子類：derived class
