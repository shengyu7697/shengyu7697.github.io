在 Visual Studio 的 Release 模式下使用中斷點

本篇記錄一下如何在 Visual Studio 的 Release 模式下偵錯時使用中斷點，停用最佳化。
專案 > 屬性 > C/C++ > 最佳化 > 停用
改完後可以發現 xxx.vcxproj 的 Optimization 標籤從 MaxSpeed 變成 Disabled，如下：

變成

之後程式再重新編譯，插入中斷點，開始偵錯F5，就可以發現程式停在中斷點上了，這樣就成功搞定囉!!!
