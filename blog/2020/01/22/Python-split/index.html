Python split 字串分割用法與範例

本篇介紹 Python 字串分割 split 的用法與範例，在字串處理中常常會需要將字串根據某個符號來切割，例如：空白，逗號，藉此區分出資料的欄位，常見於讀取 csv 的欄位切割處理，Python split 字串分割算是字串處理中最常用到的功能，

本篇內容將分為：

  • Python split 字串切割使用範例
  • split() 指定 , 逗號為分隔符號
  • split() 指定了不存在的分隔符號會怎樣？

以下範例是在 Python 3 環境下測試過。

Python split 字串切割使用範例

Python split 的用處在於把字串透過指定的分隔符號對字串進行分割，
split() 第一個參數為指定的分隔符號，可以是空白,換行(\n)，tab(\t), 逗號(,) 等等，
split() 不給入參數的話，預設會以空白作為分隔符號做切割，

split() 指定了不存在的分隔符號會怎樣？

如果 split() 指定了不存在的分隔符號的話會回傳長度為1的list，回傳的這個list第一個元素即為原本的字串。
