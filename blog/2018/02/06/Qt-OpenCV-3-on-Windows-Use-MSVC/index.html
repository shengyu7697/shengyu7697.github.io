Qt and OpenCV 3 on Windows (使用MSVC)

介紹一下怎麼在 Windows 下使用 Qt 和 OpenCV 3.

系統環境

  • Windows 7
  • Qt5.10.0 (編譯器選擇 MSVC 2015 32-bit)
  • OpenCV 3.3.1 (準備用 MSVC 2015 來 build)
  • Cmake
