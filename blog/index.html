那要怎麼去連結 libc++ 呢？

std::set 和 std::vector 有什麼不同？

  • 唯一性
    set 跟 vector 不同之處是 set 容器裡面的元素是唯一的，具有不重複的特性，vector 則沒有這個限制。
  • 不可修改性
    vector 可以修改元素的值，但 set 容器裡面元素的值是不可修改的。
  • 順序性
    set 是有序的，也就是裡面的元素會按照特定順序擺放，跟插入順序無關，vector 則不是。
