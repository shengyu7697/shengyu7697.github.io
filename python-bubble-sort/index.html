泡沫排序法 bubble sort 基本原理

泡沫排序法 bubble sort 的原理是將兩個相鄰的數值相比，假如前一個數值比後一個數值大時，就互相對調，實作時就是使用兩層迴圈，針對該串列掃兩次，最終將能獲得排序好的升序串列(由小到大)，
